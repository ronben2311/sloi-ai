// How it works — Managed Agentic Team
01
Add 4 lines to your config
Paste the SLOI AI MCP block into your Claude Desktop or Cursor config. No build step, no dependencies.
"sloiai": { "command": "npx", "args": ["@sloiai/mcp"] }
02
Set your Agent Mandate
Define your product, target price, incoterm, and auto-approve threshold. The agent negotiates within those limits without asking each time.
sloi_set_mandate({ product: "Steel Rebar", target: 2700, strategy: "aggressive" })
03
Agent runs end-to-end
SLOI AI screens counterparties, runs compliance stages, issues a compliant LOI, and streams the live log back to your IDE. You authorize or it auto-closes.
