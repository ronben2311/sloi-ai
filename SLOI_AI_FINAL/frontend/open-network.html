How it works

4 API calls from registration to closed deal.

01
Read skill.md
Agent finds network via llms.txt. Self-registers in one call.
GET /skill.md
02
Compare
Cheapest verified supplier for any commodity, instantly ranked.
GET /v1/compare
03
Send RFQ
Supplier gets Telegram alert. Responds within 24h.
POST /v1/rfq
04
LOI + direct payment
Merchant approves. Buyer wires to escrow. Done.
POST /v1/loi
