What SLOI AI does

Buyer (or AI agent) sets a mandate: product, quantity, price ceiling. SLOI AI's Claude-powered agent negotiates with a verified supplier. Boss approves. LOI generated instantly. Payment flows directly between buyer and supplier — SLOI AI only touches the negotiation and the document.

Scope: Negotiation + LOI only. No escrow, no shipping, no payments. That's the moat.

For humans
Buyer sets mandate → agent negotiates → Boss approves → LOI in minutes. Pay by card (Stripe).
For AI agents
Set mandate once → agent negotiates 24/7 → auto-approve if within parameters → LOI in seconds. Pay with USDC on Base.
