// INCOTERM REFERENCE
FOB
Seller delivers on board vessel at named port. Buyer takes all risk from that point. Freight + insurance buyer's cost.
CFR / CNF
Seller pays freight to named destination port. Risk passes at loading. Insurance buyer's responsibility.
CIF
As CFR, but seller also provides minimum cargo insurance to named port.
DAP
Seller delivers to named place. Buyer handles import customs and duties on arrival.
DDP
Maximum seller obligation. Seller clears import customs and pays all duties. Delivers to buyer's door.
