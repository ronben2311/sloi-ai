── 2. COMMODEX_ADMIN — LOI GENERATOR CONNECTED ────────────────────────────────

The LOI Generator page (loi-generator.html) is now wired into the admin flow
at two points:

  a) Approve deal button
     When boss clicks "Approve → Generate LOI", after the API call succeeds the
     admin automatically opens loi-generator.html in a new tab, pre-filled with:
       - LOI ref (from API response, or auto-generated from negotiation ID)
       - Buyer name
       - Product
       - Price
     loi-generator.html already reads these from URL params (?ref=&buyer=&product=&price=)
     and shows a green "Pre-filled from Agent Console" banner.

  b) LOI Fee Ledger table
     Each row in the Revenue → LOI Fee Ledger now has a "📄 View LOI" button.
     Clicking it opens loi-generator.html pre-filled with that LOI's ref, buyer,
     and total value. Table colspan updated from 7 → 8 to accommodate the new column.
